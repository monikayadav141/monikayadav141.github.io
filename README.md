# Monika Yadav - Personal Website

This repository hosts the personal website of Monika Yadav, a PhD student in Sociology at Columbia University.

## Development

To preview the website locally, you can use Python's built-in HTTP server.

1.  Navigate to the `monikayadav.github.io` directory in your terminal.
2.  Run the following command:

    ```bash
    python3 -m http.server
    ```

3.  Open your web browser and go to `http://localhost:8000` to see the website.
